# Best motion skills

This ranking focuses on resources that help AI coding agents add motion with taste, restraint, and product intent.

## What great motion guidance looks like

- Animation supports hierarchy and understanding
- Timing is intentional rather than flashy
- Micro-interactions reinforce state changes
- Motion is accessible and easy to tone down

## Motion evaluation checklist

Use a motion skill only if it tends to improve:

- entrance and exit clarity
- hover and focus behavior
- loading and empty states
- modal, panel, and navigation transitions
- perceived performance without distraction

Avoid motion guidance that mainly produces:

- constant floating elements
- exaggerated spring effects everywhere
- unmotivated parallax
- decorative animation with no product meaning
