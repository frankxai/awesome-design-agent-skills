# Repository Instructions

This repo is the public design-agent discovery layer.

## Scope

- Curate high-signal design skills, rubrics, examples, and rankings for AI coding/design agents.
- Own broad UI quality, anti-slop design, brand-system discovery, design audits, and before/after examples.
- Defer motion-specific rankings to `awesome-motion-design-agent-skills`.
- Defer owned brand doctrine to `starlight-design-intelligence`.

## Quality Bar

- Rank by demonstrated usefulness to shipped design quality, not by stars alone.
- Prefer resources that improve hierarchy, spacing, typography, density, accessibility, visual system coherence, and interaction polish.
- Avoid broad prompt dumps, generic inspiration lists, and resources with no visible output quality bar.

## Safety

- Keep public-safe. Do not include private brand strategy, client work, credentials, or internal-only visual systems.
- Link to official sources and active repos when possible.

## Verification

- Scan for broken links, stale dates, placeholders, and private paths before handoff.
