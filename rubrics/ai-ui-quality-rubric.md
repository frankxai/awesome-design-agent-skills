# AI UI quality rubric

A lightweight scorecard for evaluating the quality of design-oriented agent skills.

## Score each category from 1 to 5

### 1. Taste

Does the skill guide the agent toward distinctive, credible visual decisions?

### 2. Clarity

Does the resulting interface communicate structure and purpose quickly?

### 3. Consistency

Does the output hold together as a coherent system across components and screens?

### 4. Interaction detail

Does the skill improve states, transitions, and responsive behavior?

### 5. Reusability

Can the guidance be applied across different products without collapsing into sameness?

### 6. Cross-tool usefulness

Is the resource usable across multiple coding-agent ecosystems or tightly locked to one?

### 7. Audit value

Does it help critique and improve weak UIs, not just generate first drafts?

## Interpreting totals

- **30-35:** elite resource
- **24-29:** high-signal resource
- **18-23:** situationally useful
- **Below 18:** low-signal or too generic
